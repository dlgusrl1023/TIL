## vim setting

.vimrc 파일을 home에 만든다

```vim
ihyeongiui-MacBook-Pro:~ LHK$ vim ~/.vimrc
```
.vimrc 파일에 다음 setting을 해준다.

```vim
set nu                              // line number 을 표시해줌
set hlsearch                        // ‘/’ 를 눌러서 검색을 하면 검색 결과 밑에 밑줄로 하이라이트를 해
set incsearch                       // 검색어를 입력할 때마다 일치하는 문자열을 강조해서 표시
syntax enable                       // syntax에 맞게 색으로 표시를 해줌
file type plugin indent on          // file type 을 지정해주면 알맞게 연결해줌
set nocompatible                    // vim 만의 기능을 사용 할 수 있음
set mouse=a                         // mouse를 vim에서 사용 할 수 있게 해줌
nnoremap Y $y                       // 텍스트를 줄 끝으로 복사하려면 y$를 누르거나 다음을 사용하고 Y대신을 누름
set cusorline                       // 커서에 밑줄을 긋는다
```
```vim
:colorscheme                        //tap을 이용해서 color를 설정 할 수 있음
```

## vim 종료

```vim
:q!                                 // 저장하지 않고 강제종료
:wq                                 // 저장하고 종료
:w                                  // 저장하기
:cq                                 // 취소하고 종료
Shift + zz                          // 취소하고 종료
:x                                  // 변경사항이 있으면 저장하고 종료
```

## vim 점프

```vim
^	                                  // 현재 라인의 가장 첫번째 컬럼으로 점프
0	                                  // 현재 라인의 첫번째 글자로 점프
$	                                  // 현재 라인의 마지막으로 점프
w	                                  // 단어 단위로 점프
b	                                  // 역방향 단어 단위로 점프
H	                                  // 현재 화면의 첫 라인으로 점프
L	                                  // 현재 화면의 마지막 라인으로 점프
M	                                  // 현재 화면의 중간 라인으로 점프

f[문자]	                            // 오른쪽으로 검색                             
                                    예)fb : 가장 가까운 b로 점프
	    ;	다음 검색 결과로 점프
	    ,	이전 검색 결과로 점프
F[문자]	                             // 왼쪽으로 검색
            ;	다음 검색 결과로 점프
	    ,	이전 검색 결과로 점프
t[문자]	                             // f와 같지만 한칸 이전으로 간다.
            ;	다음 검색 결과로 점프
	    ,	이전 검색 결과로 점프
T[문자]	                             // F와 같지만 한칸 이전으로 간다.      
            ;	다음 검색 결과로 점프
	    ,	이전 검색 결과로 점프

[숫자]f[문자]	                    //숫자만큼 반복해서 검색.                        
                                    예)2fo : 두번째로 가까운 o로 점프
```

## vim mark

```vim
m[알파벳]		                         // mark 해줌
:marks	                            // status 볼수 있음
‘[알파벳]	                           // mark해준 곳으로 갈 수 있음
`[알파벳]	                           // mark지정해준 곳으로 커서를 옮겨줌
```

## vim 검색

```vim
/	                                  // 검색
      n	 다음 결과
      N	 이전 결과
                                        예) 2 /l 하면 커서에서 부터 2번째 l로 간다.
?	                                  // 검색 (역방향)
* 	                                // 단어 검색
#	                                  // 단어 역방향 검색                       
```

## vim 커서 이동

```vim
j	                                // 아래로 간다
k	                                // 위로 간다
^f	                              // page down
^b	                              // page up
zz	                              // 현재 커서가 있는 라인을 화면의 가운데로 스크롤
^y                                // 한 줄씩 위로 스크롤
^e                                // 한 줄씩 아래로 스크롤

gg	                              // 해당 줄로 커서 이동.	
                                  예)10gg->10번 line
                                     gg-->파일의 첫번째
G	                                // 해당 줄로 커서 이동.	
                                  예)10G-->10번 line 
                                     G-->파일의 마지막
:	                                // 해당 줄로 커서 이동.	
                                  예):10-->10번 line.
%	                                // 괄호에 커서를 두고 % 누르면 짝으로 간다.
g;	                              // 마지막으로 편집 했던 곳으로 이동가능
g,	                              // g; 와 반대의 의미
```

## vim visual mode

```vim
v	                                 // visul 모드
      0	맨처음으로 커서를 옮길 수 있다.
V	                                 // line visul 모드
^v	                               // vertical visul 모드
      커서 옮겨서 선택하고, I 누르고 앞에 적어주고 esc 눌러주면 선택된 블럭에 모두 추가된다.
```

## vim 복사 / 붙여넣기

```vim
y	                                 // 복사(yank)
yy	                               // 줄복사	
                                   예) 3yy-->3줄 복사
p	                                 // 붙여넣기
                                   예) 100p-->100번 붙여넣기
P	                                 // 커서 앞으로 붙여넣기
```

## vim 정규식

```vim
(vim 정규식)
\<	                                // 단어시작(pearl 에서 \w와 같음)
\>                                  // 단어끝
```

 \v	기본적으로 아는(perl) 정규식으로 사용가능하다.
 
```vim
/\v^.\*(vim)		->		^.\*(perl)
```
